# Personal website (GitHub Pages)

This repository is the source for my personal website. It's a simple static site:
- index.html (homepage)
- assets/style.css (styles)
- about.md, publications.md (editable content)

To publish:
1. Push these files to the repository `URFIN-Raphael/URFIN-Raphael.github.io` on the default branch (main).
2. GitHub will automatically serve the site at https://URFIN-Raphael.github.io

Preview locally:
- From the repo folder run:
  - python3 -m http.server 8000
  - then open http://localhost:8000
